Explain the difference between accessing data from arrays and objects.

The difference is that with data in an object, you can use a string to access the data within the object, but with array you have to use a numeric key. Exampe of the same information stored in an array and a non-array object:
Array: var pets = ["cat", "dog", "goldfish"]
Object: var pets = {pet1: cat, pet2: dog, pet3: goldfish}
Accessing the data in the array is done by using an index in brackets: "pets[2]" which will return "Goldfish" as we start counting at zero. Accessing the data in the object using the string: "pets.pet2" which will return "dog".
