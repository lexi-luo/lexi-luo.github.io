HTML, CSS, DOM


hmtl is the content (individual bits), css is the visual rules (structure for appearance), dom is the structure to access and manipulate content/structure/style (how these things interact)


boxifying design


everything on a website is a box, might not look like a box, but it's a box. at it's heart, still a box, consequently, not a heart, just a box


box model


content: the actual content inside the box, e.g. text. padding: the space between the content and the border of the box. border: the border of the box. margin: the space between this box and other boxes.
